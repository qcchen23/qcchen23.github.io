Text Relevance Classification for Multilingual Data

For this research project, a proposed multilingual text-based classifier that classifies web text in Chinese, French and Arabic is evaluated by comparing different modeling techniques and machine learning algorithms. The classifier is applied in the context of global biosurveillance and evaluation is based on data in the original languages. [Mentor: L. E. Charles]

Multimodal Sentiment Analysis for Voice Messaging Systems

This ongoing group project is my Senior Capstone project at the Department of Computer Science at Tulane University. The goal of this project is to use sentiment analysis to classify emotions and to detect the intensity of these emotions in multimodal data, including textual and speech data from voice messaging systems. Spatial Role Labeling Annotation

This machine learning project in Java adds an interactive funtionality to an existing project at UIUC. The spatial role annotator labels the spatial indicator, which is a word or phrase (most often a preposition) that establishes a spatial relationship between objects. A model that could create these annotations with high accuracy could have relevance for the fields of robotics, navigation, and the like. [Mentor: P. Kordjamishidi; Group member: K. Roberts]

New View of MultiTree

MultiTree, a three-year project at Indiana University Bloomington, is creating a digital library of scholarly hypotheses about language relationships and subgroupings. This information is being organized in a searchable database with a web interface, and each hypothesis is presented graphically as an interactive hyperbolic display of a family tree.

LL-MAP Interface

LL-MAP (Language and Location - A Map Annotation Project) is a project at Indiana University Bloomington designed to integrate language information with data from the physical and social sciences by means of a Geographical Information System (GIS). I worked with a team of interns to implement the new interface for the LL-MAP site.
